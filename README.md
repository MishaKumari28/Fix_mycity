FixMyCity 🏙️ — AI-Powered Civic Issue Reporting

Built with Python Flask + IBM Watsonx.ai (Granite) · Responsive Bootstrap UI · Indian government integrations

✨ Features:-

- AI Classification-	IBM Granite-3-8B classifies civic issues from text + image.
- mage Upload-	File upload or live camera capture with drag-and-drop.
-> Formal Complaint- Generator	AI drafts a professional Indian government complaint letter.
-> Department Routing-	Matches issue + location → correct ministry/portal/helpline.
-> Safety Gate-	Sensitive issues (DV, child abuse) → confidential text-only helpline flow.
-> Geolocation-	GPS detection + reverse geocoding via Nominatim.
-> Complaint Tracker-	Track status by complaint ID.
-> Dashboard + Map-	Category charts, Leaflet map of geotagged reports.
-> Dark Mode-	System-aware + manual toggle.
-> AGENT_INSTRUCTIONS-	Single dict in app.py to customise everything.
